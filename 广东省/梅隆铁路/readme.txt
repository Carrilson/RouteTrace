贡献者：香茶

---------------

所经城市：梅州

---------------

更新时间：2018.4.10

---------------

简介：梅隆铁路是由梅四、兴四、兴老三条铁路连接而成，以运煤为主、运客为辅的762毫米轨距窄轨铁路，起自梅县(今梅州市区)东山港，经兴宁、五华，到达龙川县老隆港，正线长172.38公里，共设有30个车站：东山、梅县(大浪口)、罗坑、长滩、赤水、三门缺、上官塘、车子排、白面、均田、西埔、黄陂、甘砖、岗背、土墩、龙北、合水、龙田、兴城、红光、汤湖、高洋寨、莲池、洋田、铁场、谷前、河坑、莲风、丰稔、老隆，其中西埔站为编组站，各车站到发线股道有效长度在110～270米之间。此外，还有四望嶂、黎树坑、石壁塘等装卸点。

自1958年11月梅四线开工起，到1970年10月兴老线建成止，梅隆铁路前后历时12年，全部修建费用为8841.6万元。当年的梅隆铁路，无论在技术上还是运营能力上，在广东地区可算是首屈一指。兴宁四望嶂煤矿曾是广东省最大的煤矿之一，为解决四望嶂煤炭外运和汕头地区工业、民用煤炭供应，从梅县东山港至四望嶂矿区全长62.6公里的梅四铁路应运而生。该铁路1964年10月建成通车。

梅四铁路按年运量60万吨的能力进行设计，铺每米15公斤钢轨，以煤矿坑木作轨枕，机车为本省制造的“上游”型蒸汽机车，车辆为载重6～10吨的手制动翻斗车，车站到发线股道长度不到百米，无信号设备，由人工引导进站。1970年，梅四、兴四铁路有“上游”型蒸汽小机车8台，各类车辆147辆(其中客车8辆)。梅隆铁路管理局统一管理后，陆续从本省三水机车车辆工厂和外省工厂购进几批，到1987年，机车增至25台(其中GZ380液力传动内燃机车11台)，各类车辆增至375辆(其中YZ21型客车16辆)。

梅隆铁路设备技术标准虽低，但运量比汽车大，运费比汽车低，它源源不断地将四望嶂的煤炭运往珠三角和粤东地区，对四望嶂煤矿业的发展，对梅州地区的经济社会发展，起到了重要作用。全盛时期整个梅隆铁路，共有员工四五千人之多，拥有自己独立的学校、医院，带动了驻地西埔甚至兴宁市的繁荣。

2005年，在各种综合因素影响下，梅隆铁路结束其历史使命，退出了历史舞台。到拆除之前为止,梅隆铁路属广东省地方铁路总公司梅隆铁路管理处管理。现在，梅隆铁路已经全线拆除了。
