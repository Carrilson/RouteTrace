贡献者：威，香茶

---------------

所经城市：坪石，连州

---------------

更新时间：2018.4.21

---------------

简介：

1958年，广东省决定开发连县煤炭资源。为解决运输问题，铁道部和广东省联合投资修建韶(关)柳(州)铁路的坪石至连县路段(当时称坪连铁路)106公里。因坪石三星水库储水标高尚未定案，该路自坪石站西行有24公里线路选线不能最后确定，暂利用南岭支线铁路作过渡便线，于岩泉(今新团结站)与坪连铁路接轨；同时投资169万元，按坪连铁路技术设备的标准，对南岭支线铁路进行技术改造。1958年9月，铁道第四勘察设计院编制了韶柳线简明初步设计，韶关至连县（现连州市）段研究了在坪石、韶关、沙口等站接轨的方案，10月经铁道部批准。1959年11月，新团结站至连州段开始施工，同时进行南岭支线改造工程。至1961年9月因压缩基建停工；当时已完成工程投资3735万元，占预算8864万元的42％。
