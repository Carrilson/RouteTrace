简介：七滦线曾经是津山铁路（原京山线）正线的一部分，是中国自建的最早的铁路之一。其中七滦线上的唐山南――胥各庄段，就是著名的唐胥铁路，是中国自主建设的最早的铁路。从上个世纪80年代起，由于开滦煤矿长期对地下煤层开采过度，导致地下出现巨大的空洞，七滦线的部分路基出现路基下沉，逐渐严重影响到列车行驶安全，有时无法正常通行客车。
于是，铁道部决定从七道桥上行站唐坊站到滦县再修一段新的京山铁路，绕过已经处于危机之中的七滦线即“京山压煤改线”工程。1994年，京山压煤改线工程完工，“压煤改线”的路段成为京山铁路的正线，从七道桥到滦县的老京山线改称“七滦线”，成为支线铁路，从此辉煌不再。
